What is it/how does it work?

It is a type of formula used whenever you have something like f=ma. You have a top, left and right number that work like this:
- To find the value of the top number, multiply the bottom numbers(left x right).
- To find the value of the right number, divide the top number by the left number.
- To find the value of the left number, divide the top number by the right number.
